# 📊 Swiggy Sales Dashboard – Excel Project

## 📌 Project Overview
This project is an interactive Sales Dashboard built in Microsoft Excel to analyze Swiggy food delivery data.

The dashboard provides insights into:
- Total Sales
- Average Rating
- Average Order Value
- Total Orders
- Monthly & Daily Trends
- Quarterly Performance
- Top 5 Cities
- Regional Sales Distribution (India Map)

---

## 🛠 Tools Used
- Microsoft Excel
- Pivot Tables
- Pivot Charts
- Slicers
- Conditional Formatting
- Map Chart

---

## 📈 Key Insights
- Total Sales: ₹53.01M
- Total Orders: 197K+
- Average Rating: 4.34
- Highest Performing City: Bengaluru
- Strongest Quarter: Q2
